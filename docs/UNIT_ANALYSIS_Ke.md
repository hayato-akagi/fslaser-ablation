# Ke (電子熱伝導率) の単位変換解析

## 論文の記載

### 引用元 [19] の論文
```
Ke = −3.47 ×10^8 + 4.45 ×10^6 Te  [eV/sA K]
```

### JMMP-07-00068 (Liu & Cheng, 2023)
```
Ke = 1.6 ×10^11 × (−3.47 ×10^8 + 4.45 ×10^6 Te)  [W/cm K]
```

---

## 単位の解釈

### 可能性1: `sA` = `s·A` (秒・アンペア)

`[eV/(s·A·K)]` → `[W/(cm·K)]` への変換

- 1 eV = 1.602×10^-19 J
- 1 W = 1 J/s = 1 eV/s × 1.602×10^-19
- A (アンペア) = 電流の単位

**問題点:** 
- 熱伝導率に「電流」の単位が入るのは物理的に不自然
- `eV/(s·A·K)` は `[J·s^-1·A^-1·K^-1]` = `[V·K^-1]` (電圧/温度)
- 熱伝導率の次元 `[W/(cm·K)]` = `[J·s^-1·cm^-1·K^-1]` と合わない

**結論:** この解釈は不適切

---

### 可能性2: `sA` = `s·Å` (秒・オングストローム)

`[eV/(s·Å·K)]` → `[W/(cm·K)]` への変換

- 1 Å = 10^-10 m = 10^-8 cm
- 1 eV = 1.602×10^-19 J

変換係数:
```
1 [eV/(s·Å·K)] = 1.602×10^-19 [J] / (1 [s] × 10^-8 [cm] × 1 [K])
                = 1.602×10^-19 / 10^-8 [J/(s·cm·K)]
                = 1.602×10^-11 [W/(cm·K)]
```

**引用元の式を変換:**
```
Ke = (−3.47×10^8 + 4.45×10^6 Te) [eV/(s·Å·K)]
   = (−3.47×10^8 + 4.45×10^6 Te) × 1.602×10^-11 [W/(cm·K)]
```

**Liu論文が係数 1.6×10^11 を掛けた式:**
```
Ke = 1.6×10^11 × (−3.47×10^8 + 4.45×10^6 Te) [W/(cm·K)]
```

**比較:**
- 引用元を変換: `... × 1.602×10^-11` [W/(cm·K)]
- Liu論文: `1.6×10^11 × ...` [W/(cm·K)]

**差分:** `1.6×10^11 / 1.602×10^-11 = 1.0×10^22` 倍の違い！

**結論:** Liu論文は単位変換の指数部の正負を間違えた可能性

---

### 可能性3: `A` = `Å` (オングストローム) の略記

`[eV/(s·Å·K)]` と同じ解釈 → 可能性2と同じ結論

---

### 可能性4: 引用元の単位表記ミス

引用元が `[eV/(s·cm·K)]` と書くべきところを `[eV/sA K]` と誤記した場合:

```
1 [eV/(s·cm·K)] = 1.602×10^-19 [J/(s·cm·K)]
                 = 1.602×10^-19 [W/(cm·K)]
```

この場合:
```
Ke = (−3.47×10^8 + 4.45×10^6 Te) × 1.602×10^-19 [W/(cm·K)]
```

Liu論文が `1.6×10^11` を掛けると:
```
Ke ≈ 2.56×10^-8 × (−3.47×10^8 + 4.45×10^6 Te) [W/(cm·K)]
```

Te=1000K で Ke ≈ 1.09×10^-1 W/(cm·K) → 妥当な値

**結論:** この解釈でも `10^11` は大きすぎる

---

## 物理的妥当性チェック

### 典型的な熱伝導率の値

| 物質 | 熱伝導率 [W/(cm·K)] |
|------|---------------------|
| 銅（室温） | 3.85 |
| アルミ（室温） | 2.37 |
| シリコン（室温、格子） | 1.48 |
| 液体シリコン | 0.5-0.7 |

### 計算結果の比較

#### ケース A: Liu論文の式そのまま（係数 = 1.6×10^11）
```
Te = 300K:  Ke = 1.58×10^20 W/(cm·K)  ← 銅の 10^19 倍！
Te = 1000K: Ke = 6.57×10^20 W/(cm·K)
```
**結論:** 物理的に不可能な値

#### ケース B: 単位変換を適用（係数 = 1.6×10^-11）
```
Te = 300K:  Ke = 1.58×10^-2 W/(cm·K)  ← 妥当
Te = 1000K: Ke = 6.57×10^-2 W/(cm·K)  ← 液相(0.67)に近づく ✓
```
**結論:** 物理的に妥当な値

## 推定される原因

### 最も可能性が高い原因

**Liu et al. が単位変換時に指数の正負を間違えた**

引用元:
```
Ke = (−3.47×10^8 + 4.45×10^6 Te) [eV/(s·Å·K)]
```

正しい変換:
```
係数 = 1.602×10^-11 [W/(cm·K)] per [eV/(s·Å·K)]
Ke = (−3.47×10^8 + 4.45×10^6 Te) × 1.602×10^-11 [W/(cm·K)]
```

Liu論文の記載:
```
Ke = 1.6×10^11 × (−3.47×10^8 + 4.45×10^6 Te) [W/(cm·K)]
```

**10^-11 が 10^11 に変わっている（指数の正負が逆）**

## 推奨アクション

### 実装上の判断

**1.6×10^-11 を採用すべき理由:**

1. **物理的妥当性** - 液相値(0.67)との連続性 ✓
2. **単位変換の整合性** - eV/(s·Å·K) → W/(cm·K) の変換と一致
3. **数値安定性** - CFL条件を満たすdtが現実的な範囲

**1.6×10^11 を採用できない理由:**

1. **物理的不可能** - 金属の10^19倍の熱伝導率
2. **相間の不連続** - 液相(0.67)から固相(10^20)への飛躍
3. **数値不安定** - CFL条件が dt < 10^-22 s を要求（非現実的）

## まとめ

**結論:** Liu論文の `1.6×10^11` は**指数の正負ミス**と判断されます。

**根拠:**
- 単位変換: eV/(s·Å·K) → W/(cm·K) は 1.602×10^-11 倍
- 物理的妥当性: 10^-11 は○、10^11 は×
- 相転移の連続性: 10^-11 は液相と整合、10^11 は不整合

**推奨:** 実装には **1.6×10^-11** を使用
